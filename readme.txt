APR calculator that uses the formula APR = ((I + F) / P) / N can help borrowers understand the true cost of loans and how to calculate APR on a loan.

[https://www.calculator.io/apr-calculator/](https://www.calculator.io/apr-calculator/)

== Usage ==

Add the APR Calculator shortcode to your page, post or sidebar:

`[ci_apr_calculator]`

Add the following code to your template where you would like the APR Calculator to appear:

`<?php display_ci_apr_calculator(); ?>`

== Installation ==

1. Upload the APR Calculator /ci_apr_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the APR Calculator plugin through the "Plugins" menu in WordPress.
